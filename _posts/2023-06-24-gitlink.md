---
Git Bash 콘솔창 설정
```
git config --global user.name "사용자 이름"
```
깃허브 사용자 이름 쓰기
```
git config --global user.email "사용자 이메일"
```
깃허브 이메일 쓰기
```
git config --list
```
확인 리스트

VSCode 터미널 설정
```
git init
```
초기화
```
git add .
```
내용을 전부 보내겠다.
```
git commit -m "first commit"
```
메모리 이름 설정
```
git remote add origin https://github.com/duswntmd/폴더.git //폴더부분 변경
```
내 git주소로 폴더 연결
```
git remote -v
```
연결 확인
```
git push origin master
```
github에 업로드 완료
