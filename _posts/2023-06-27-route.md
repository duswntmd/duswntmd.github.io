title:  "route 중요성"
---
백엔드를 사용하면서 주소의 고민을 많이 했었다.
주소의 경로에 따라 홈페이지 모습을 만드는 것을 꺠달았고 필요한 것들을 경로에 모아 만들으면 된다.
```
app.get('/', (req, res) =>
{
안의 내용 .. 
}
```
작성을 할 경우 localhost본인이 정한 포트번호 /없이 눌렀을 경우 나온다.
만약 아래 코드처럼 작성 했다면
```
app.get('/create', (req, res) =>
{
안의 내용 .. 
}
```
localhost본인이 정한 포트번호 뒤에 /를 추가한 후 create주소를 치면 나오는 것을 볼 수 있다.
