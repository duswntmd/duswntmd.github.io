title:  "listen의 여러포트 사용"
---
js파일을 돌려보다 포트이동을 사용하여 다른포트로도 사용가능한가의 연구를 하고 있었다.
```
app.listen(3000, () => {
  console.log(`server on 3000`);
});
```
app는 express를 넘겨 받은것이다.
하나의 js파일을 3000으로 돌리고 있다.
하지만 다른 파일을 불러올 경우 그 파일 안에 있는 listen를 따로 사용할 수 있는것 같다.
```
var 사용할 변수명 = require('./불러올파일이름.js');
```
이런식으로 터미널로 돌아가는 파일에 불러올시 포트가 2개 돌아가는 것을 확인 했다.
